#  Auto Mileage Prediction

A Machine Learning based web application that predicts the mileage of a car based on user inputs.

 Project Overview
This project uses a trained Machine Learning model to estimate car mileage (MPG).  
It is integrated with a Flask web application where users can enter car details and get instant predictions.

 Features
- User-friendly web interface
- Real-time mileage prediction
- Machine Learning pipeline
- Data preprocessing included
- Simple and clean UI

 Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML/CSS



 How It Works
1. User enters car details in the form
2. Data is processed using Pandas & NumPy
3. Trained ML model predicts mileage
4. Result is displayed on the webpage
